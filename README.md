# SPloiter
A Tool used for Three Purposes:-
ReverseIp Lookup,
BruterForcing(FTP,SSH),
Subdomain Enumeration

# Installing
Open Terminal and Type:
```
git clone https://github.com/Nadar-Athis/SPloiter
```
```
cd SPloiter
```
```
ls
```
```
chmod +x *
```
```
pip install requirements.txt
```
```
python3 SPloiter.py
```
